## 2024-04-12 - Optimize Dictionary Iteration inside SendThread and processPayload loops
**Learning:** In high-frequency loops, iterating over dictionaries by explicitly fetching key-value pairs via `.items()` is marginally faster (and more readable) than iterating over the keys alone and executing an extra dictionary key lookup, because it avoids redundant hash computations. However, caching deep dictionary lookups to a local variable does not yield similar improvements in Python.
**Action:** When repeatedly iterating over dictionary keys to access values in hot loops, use `.items()` instead. Always document optimizations using inline code comments, particularly when instructed to do so.
